Objects

Obejcts are to classes what houses are to blueprints

Simply but, an object is an instance of a class...

Python is an object-oriented programming language. An object is simply a collection of data (variables) and methods (functions) that act on those data.

We can create new object instances of classes. The procedure to crate it is similar to a function call.

harry = Person()

This will create a new object instance named harry. We can access the atributes of objects using the object name prefix.

It means to say, since Person.greet is a function objects (attribute of class), Person.greet will be a method object.

class Person:

"This is a person class"

age = 10

def greet(self):

print('Hello')

# create a new object of Person class

harry = Person()

# Output: function Person.greet

print(Person.greet)

# Output: bound method Person.greet of __main__.Person object

print(harry.greet)

# Calling object's greet() method # Output: Hello

harry.greet()
