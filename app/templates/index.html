Differences between an Object and a Class
Difference # Object Class
1 Object is an instance of a class. Class is a bluprint or template from which objects are created.
2 An object is a physical entity. Class is a logical entity.
3 Object is allocated memory when created. Class does not allocate any memory when created.
4 Object is a specific entity. Class is a group of identical objects.
5 A class can have any number of instances or objects. The class can be declared only once.
